# CNN Architectures

哈工大(深圳)计算机视觉课程项目



## 项目概述

​	该项目的结构为

```
cnn_architectures
	|-AlexNet
		|-AlexNet.md	#AlexNet网络介绍
		|-train.py		#训练文件
	|-GoogleNet
		|-model.py	#模型文件
		|-train.py # 训练文件
	|-ResNet
		|-train.ipynb	
	|-VGG
		|-train.ipynb	#训练文件 (jupyter Notebook)
		|-VGG.md
	
```

### 环境要求

​	`python >= 3.6`

​	`pytorch >= 1.2.0`

### 使用方法

#### train.py

​	若训练文件为train.py文件:

​	修改``train.py``中数据集的地址。

​	`python train.py`

#### train.ipnyb

​	输入`jupyter notebook`

​	修改数据集的地址

​	运行即可
